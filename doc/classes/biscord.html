# Biscord

Construct Biscord framework for your Discord Bot.

# Installation


  # Install

  npm install biscord

  # Update

  npm update biscord

# New Instance


  var biscord = require('biscord');

  var Biscord = new biscord( Token , ClientOptions );
Parameter Optional Description
Token (opens new window) false Your Discord Bot application token.
ClientOptions (opens new window) true Client Options for Discord Bot Client.

Returns: Client Object (opens new window)

# Initializing

WARNING

Before you intialize your Client, make sure the managers you wish to use is constructed.

To add Managers visit the Managers section.

Initialize your Biscord Client to put in online.


  Biscord.intialize();

Returns: Promise (opens new window)<Boolean , Client>

# Event Methods

Biscord supports Event Listener and Emitter, read more at Events.

Event Parameters Descripiton
ready None The client is ready.
destroy None The client was destroyed.
